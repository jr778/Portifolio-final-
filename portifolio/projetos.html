Projetos de Segurança da Informação

App de Segurança da Informação

Aplicativo seguro para armazenamento de dados sensíveis usando Django + Cryptography. Sistema completo com criptografia Fernet, autenticação de usuários e dashboard protegido.

Python Django Cryptography PostgreSQL Fernet

Funcionalidades:

  • 🔐 Criptografia avançada com Fernet
  • 👤 Sistema de autenticação Django
  • 💾 Armazenamento seguro de dados sensíveis
  • 📊 Dashboard protegido por login
  • 🔑 Geração automática de chaves criptográficas
  • 📝 Interface para adicionar/visualizar dados

Estrutura do Projeto:

# Instalação
pip install django cryptography psycopg2-binary

# Estrutura
app_seguranca/
├── seguranca/
│   ├── models.py      # Modelo DadosSensiveis
│   ├── forms.py       # Formulários
│   ├── views.py       # Views protegidas
│   └── urls.py        # Rotas
├── templates/
│   ├── dashboard.html # Dashboard principal
│   └── adicionar.html # Form de adição
└── settings.py        # Configurações Django
